# Tahir Saeed Portfolio - Plain HTML/CSS/JS Version

This is a plain HTML, CSS, and JavaScript version of the Tahir Saeed portfolio website.

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `script.js` - Interactive functionality

## How to Use

1. Simply open `index.html` in your web browser
2. Or use a local server:
   - Using Python: `python -m http.server 8000`
   - Using Node.js: `npx http-server`
   - Using PHP: `php -S localhost:8000`

## Features

- Fully responsive design
- Dark theme with modern aesthetics
- Interactive workflow section
- Smooth scrolling navigation
- Hover effects and transitions
- No dependencies or build tools required

## Customization

- Edit `index.html` to change content
- Modify `styles.css` to adjust colors and styling
- Update `script.js` for different interactions

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
